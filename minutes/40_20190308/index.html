    第40回議事録

    概要

    • 2019/03/08 8:00-9:30
    • 参加者：栗田、小渕、佐藤、鈴木、荻生、川畑、岡本、楠、中島、林、須賀、上原

    agenda

    1. Milestone Overview
    2. Tasks & Issues
    3. Status update
    4. Security WG
    5. Inventories
    6. Tasks and To-Dos
    7. AOB / Housekeeping

    議事サマリ

    agenda1: Milestone Overview

    • JVCEA技術委員会
      3月5日に実施。CGTFドキュメントから作成したチェックリスト案の議論、公表文案、用語の定義が示された。今後、提示文については引き続きで議論し、次回（19日）に最終決定予定。
    • 金融庁
      来週面談予定。

    agenda2: Tasks & Issues

    • 日本国内における仮想通貨ウォレット業務の実態調査 レビュー中
    • 39回までの議事録承認済み。公開に向けて準備予定。

    agenda3: Status update

    • JVCEA
      議事録、資料の共有
    • 金融庁
      来週、カストディアンの実態調査報告予定
    • 事務局
      構成員決定。今後、事務局とBOT、CGTFChairの権限分けを明確化
      ※監査役については変更の可能性あり。

    agenda4: Security WG

    • 基準案６項目を作成。今後SecWGでブラッシュアップ
      1. 暗号資産カストディアンのセキュリティ対策についての考え方
      2. General Security Considerations for Crypto Assets Exchange
      3. Terminology for Crypto Asset
      4. Guideline 案
      5. 日本国内における仮想通貨ウォレット業務の実態調査
      6. 対策案

    agenda5: Inventories

    • セキュリティに関する基準案を3月末までに作成予定。
    • 警察連携、ホット/コールドウォレットの定義、標準化動向、会計及び税務処理に関する議論を再始動予定。

    agenda6: Next Meeting

    • 2019/03/22 08:00-09:30 TF Meeting @ 東京ミッドタウン日比谷
    • 2019/04/05 08:00-09:30 TF Meeting @ 東京ミッドタウン日比谷
